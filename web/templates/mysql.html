Scenario # 1 Setting Up Mysql Server

Base OS : Ubuntu 18.04 LTS

Environment : Docker Engine 18+

We would like you setup Mysql server with below requirements :

  • Install Mysql Server
  • Server should be available with private IP
  • Default DB engine should be InnoDB
  • One Database with name 10PearlsDB
  • One Master user that can only access 10pearlsDB with admin privileges
  • One user that has access to 10PearlsDB with readonly permissions
  • Set InnoDB engine for File per table
